Sobre o Projeto AluraGeek

O AluraGeek é um projeto de Front-End desenvolvido para uma plataforma fictícia de e-commerce focada em produtos voltados para o público geek. O projeto tem como objetivo fornecer um ambiente de aprendizado prático para o desenvolvimento de aplicações web utilizando tecnologias modernas, como HTML, CSS e JavaScript.

A interface do AluraGeek permite que os usuários visualizem produtos disponíveis no catálogo, adicionem novos itens, e excluam produtos da lista. Tudo isso de forma intuitiva, com um design limpo e responsivo, focado na experiência do usuário.

Principais Funcionalidades:

  • Catálogo de produtos exibido dinamicamente
  • Formulário para adicionar novos produtos
  • Botão para remover produtos do catálogo
  • Integração com backend para persistência de dados (opcional)

Tecnologias Utilizadas:

  • HTML5 e CSS3 para estruturação e estilização
  • JavaScript para manipulação dinâmica do DOM
  • Font Awesome para ícones
  • Node.js com Express
